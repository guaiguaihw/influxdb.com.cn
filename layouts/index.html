InfluxDB Use Cases

  • Metrics

    Pair InfluxDB with Grafana, CollectD, Sensu, or cAdvisor to roll your own custom DevOps solutions. You'll have full visibility into your infrastructure, servers, applications, and Docker containers.

  • Real-time Analytics

    Track user and business analytics in real-time with InfluxDB. Use a client library in your favorite language to track events and numbers and visualize it all with realtime dashboards on top of Grafana.

  • Sensor Data

    InfluxDB is designed to track data from tens of thousands of sensors all sampling at rates of once a second or more. Using the Influx JavaScript library and your favorite charting library, you can build custom sensor analytics in a few hours.

Features

  • Time-Centric Functions

    Everything in InfluxDB is a time series that you can perform standard functions on like min, max, sum, count, mean, median, percentiles, and more.

  • Scalable Metrics

    Collect your data on any interval and compute rollups on the fly later. Track 100 metrics or 1 million â€“ InfluxDB is designed to scale horizontally.

  • Events

    InfluxDB's data model supports arbitrary event data. You can write in a hash of associated data and count events, uniques, or grouped columns on the fly.

  • Native HTTP API

    Our built-in API makes it easy to read and write data from JavaScript. Build custom analytics frontends without writing a single line of server-side code.

  • Powerful Query Language

    A SQL-like query language designed for working with time series and analytics. Create and run queries without needing a programmer around.

  • Built-in Explorer

    InfluxDB ships with a built-in interface for querying and visualizing your data. You can manage databases, users, and permissions, too.
